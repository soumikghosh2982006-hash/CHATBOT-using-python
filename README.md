# 🤖 Basic Chatbot Using Python 3

A simple rule-based chatbot built using **Python 3**. This project demonstrates the basics of chatbot development by using conditional statements and predefined responses to interact with users.

## 📌 Project Overview

This is a beginner-friendly chatbot that can communicate with users through the command line. The chatbot recognizes common user inputs like greetings, questions, and exit commands, then responds with suitable predefined messages.

The project is created using only **Python 3** without any external libraries.

## ✨ Features

- 💬 Simple text-based conversation
- 👋 Responds to greetings
- ❓ Answers basic predefined questions
- 🔄 Continuous interaction until the user exits
- ⚡ Lightweight and beginner-friendly
- 🐍 Built completely with Python 3

## 🛠️ Technologies Used

- **Python 3**

## 🚀 How to Run the Project

### 1. Clone the repository

git clone https://github.com/your-username/Basic-Chatbot.git 

### 2. Navigate to the project folder

cd Basic-Chatbot

### 3. Run the chatbot

python chatbot.py
